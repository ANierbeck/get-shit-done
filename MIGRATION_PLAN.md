## 📋 Migration Workflow (Per Skill)

### 1. Preparation (10-15 min)
```bash
# Review original command
read_file commands/gsd/{skill}.md

# Check current skill state
read_file skills/gsd/{skill}/SKILL.md

# Note complexity and special requirements
```

### 2. Content Extraction (30-60 min)
```bash
# Extract frontmatter
grep "name:" commands/gsd/{skill}.md
grep "description:" commands/gsd/{skill}.md
grep -A 5 "allowed-tools:" commands/gsd/{skill}.md

# Extract main sections
grep -A 20 "<objective>" commands/gsd/{skill}.md
grep -A 20 "<when_to_use>" commands/gsd/{skill}.md
grep -A 100 "<process>" commands/gsd/{skill}.md
grep -A 20 "<success_criteria>" commands/gsd/{skill}.md
grep -A 20 "<anti_patterns>" commands/gsd/{skill}.md
grep -A 10 "<output>" commands/gsd/{skill}.md
```

### 3. Skill Creation (60-120 min)
```bash
# Create comprehensive SKILL.md
cat > skills/gsd/{skill}/SKILL.md << 'EOF'
---
name: {skill}
description: [extracted description with usage context]
license: MIT
metadata:
  author: get-shit-done
  version: "1.0"
  category: project-management
allowed-tools: [extracted tools]
---

# {skill} Skill

## Objective

[extracted objective with proper formatting]

## When to Use

[extracted when_to_use or created usage guidance]

## Process

[extracted process with all bash commands preserved]

## Success Criteria

[extracted success_criteria with proper formatting]

## Anti-Patterns

[extracted anti_patterns or created guidelines]

## Output

[extracted output or created documentation]

## Examples

### Example 1: [Common Use Case]
```
Input: [example input]
Output: [expected output]
```

### Example 2: [Edge Case]
```
Input: [edge case input]
Output: [handled output]
```

## Error Handling

- [Common error]: [recovery procedure]
- [Edge case]: [handling approach]
- [Missing data]: [fallback behavior]
EOF
```

### 4. Quality Assurance (15-30 min)
```bash
# Check against migration checklist
# Verify all bash commands preserved
# Test formatting and readability
# Ensure no information lost
```

### 5. Commit (5 min)
```bash
git add skills/gsd/{skill}/SKILL.md
git commit -m "feat: fully migrate {skill} skill"
  -m "Preserve all original command functionality"
  -m "Add comprehensive examples and documentation"
  -m "Ensure all bash commands and logic preserved"
```
